# 📖 Quran Website

A modern and responsive Quran reading platform built to help Muslims read, explore, and learn the Holy Quran online. The website provides a clean user interface, easy navigation between Surahs, and a smooth reading experience across all devices.

🌐 **Live Demo:** https://quran-website-ochre.vercel.app/

---

## ✨ Features

* 📖 Read all Quran Surahs
* 🔍 Search Surahs quickly
* 📱 Fully Responsive Design
* 🌙 Clean and Modern UI
* ⚡ Fast Loading Performance
* 🎨 Beautiful Typography
* 📚 Easy Navigation
* 🔖 User-Friendly Reading Experience

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6+)
* Vercel (Deployment)
